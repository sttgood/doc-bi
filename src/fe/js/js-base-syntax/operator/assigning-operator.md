用于将值赋给变量。

| 运算符 | 描述       | 示例                    |
| ------ | ---------- | ----------------------- |
| `=`    | 赋值       | `let x = 10;`           |
| `+=`   | 加后赋值   | `x += 5;` → `x` 为 `15` |
| `-=`   | 减后赋值   | `x -= 3;` → `x` 为 `12` |
| `*=`   | 乘后赋值   | `x *= 2;` → `x` 为 `24` |
| `/=`   | 除后赋值   | `x /= 4;` → `x` 为 `6`  |
| `%=`   | 取余后赋值 | `x %= 5;` → `x` 为 `1`  |
| `**=`  | 指数后赋值 | `x **= 2;` → `x` 为 `1` |
