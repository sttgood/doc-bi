| 运算符 | 描述           | 示例                  |
|--------|----------------|-----------------------|
| `,`    | 逗号运算符     | `let x = (1, 2, 3);` → `x` 为 `3` |
| `?.`   | 可选链         | `obj?.prop` → 如果 `obj` 为 `null` 或 `undefined`，返回 `undefined` |
| `??`   | 空值合并       | `let x = null ?? 'default';` → `x` 为 `'default'` |
