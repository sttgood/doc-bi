用于比较两个值，返回布尔值（`true` 或 `false`）。

| 运算符 | 描述               | 示例                  |
|--------|--------------------|-----------------------|
| `==`   | 等于（值相等）     | `'5' == 5` → `true`   |
| `===`  | 严格等于（值和类型） | `'5' === 5` → `false` |
| `!=`   | 不等于（值不等）   | `'5' != 5` → `false`  |
| `!==`  | 严格不等于         | `'5' !== 5` → `true`  |
| `>`    | 大于               | `6 > 3` → `true`      |
| `<`    | 小于               | `3 < 6` → `true`      |
| `>=`   | 大于等于           | `6 >= 6` → `true`     |
| `<=`   | 小于等于           | `3 <= 6` → `true`     |
