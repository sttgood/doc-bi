用于对二进制位进行操作。

| 运算符 | 描述           | 示例                  |
|--------|----------------|-----------------------|
| `&`    | 按位与         | `5 & 1` → `1`         |
| `|`    | 按位或         | `5 | 1` → `5`         |
| `^`    | 按位异或       | `5 ^ 1` → `4`         |
| `~`    | 按位非         | `~5` → `-6`           |
| `<<`   | 左移           | `5 << 1` → `10`       |
| `>>`   | 右移           | `5 >> 1` → `2`        |
| `>>>`  | 无符号右移     | `-5 >>> 1` → `2147483645` |
