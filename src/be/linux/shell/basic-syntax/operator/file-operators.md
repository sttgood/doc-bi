用于检查文件属性（在 `[[ ]]` 或 `[ ]` 中使用）：

| 运算符 | 描述              | 示例                   |
| :----- | :---------------- | :--------------------- |
| `-e`   | 文件/目录存在     | `[[ -e "file.txt" ]]`  |
| `-f`   | 是普通文件        | `[[ -f "file.txt" ]]`  |
| `-d`   | 是目录            | `[[ -d "/path" ]]`     |
| `-s`   | 文件大小大于0字节 | `[[ -s "file.txt" ]]`  |
| `-r`   | 文件可读          | `[[ -r "file.txt" ]]`  |
| `-w`   | 文件可写          | `[[ -w "file.txt" ]]`  |
| `-x`   | 文件可执行        | `[[ -x "script.sh" ]]` |
| `-L`   | 是符号链接        | `[[ -L "link" ]]`      |
