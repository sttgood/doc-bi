用于二进制位操作（在 `$(( ))` 中使用）：

| 运算符 | 描述     | 示例                                    |      |      |      |
| :----- | :------- | :-------------------------------------- | ---- | ---- | ---- |
| `&`    | 按位与   | `$((5 & 3))` → 1 (0101 `&` 0011 = 0001) |      |      |      |
| `|`    | 按位或   | `$((5 | 3))` → 7 (0101`|`0011 = 0111)   |      |      |      |
| `^`    | 按位异或 | `$((5 ^ 3))` → 6 (0101 `^` 0011 = 0110) |      |      |      |
| `~`    | 按位取反 | `$((~5))` → -6（补码运算）              |      |      |      |
| `<<`   | 左移     | `$((5 << 2))` → 20 (0101 → 10100)       |      |      |      |
| `>>`   | 右移     | `$((16 >> 2))` → 4 (10000 → 00100)      |      |      |      |
