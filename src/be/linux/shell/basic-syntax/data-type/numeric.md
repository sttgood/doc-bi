## 整数基础

Bash支持整数运算，但不支持浮点数。整数可以是十进制、十六进制（以`0x`或`0X`开头）或八进制（以`0`开头）。Bash中的整数运算可以使用`let`命令、双括号`(( ))`、或`expr`命令等。

- **使用`let`命令**：

  ```bash
  let num=10+20
  ```

- **使用双括号`(( ))`**：

  ```bash
  num=$((10 + 20))
  ```

- **使用`expr`命令**：

  ```bash
  num=$(expr 10 + 20)
  ```
